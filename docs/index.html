Features

Everything needed for daily SSH work.

Keep terminals, file transfer, tunnels, keys, and reusable commands in one native desktop app.

Multi-session SSH tabs

Open, duplicate, and switch between remote SSH sessions and local terminals quickly.

SFTP file transfer

Browse files, drag and drop uploads, resolve conflicts, and track transfer progress.

Port forwarding

Save local and remote SSH tunnels for databases, dashboards, and internal services.

SSH key management

Import and organize SSH keys, host fingerprints, and connection authentication material.

Snippets

Save common commands and run them across active sessions without retyping.

Cloud sync

Sync connections, keys, and settings across devices with account-based backup.

Why native

A lighter alternative to Electron SSH clients.

Noverterm vs Electron SSH clients

Tauri keeps the app lightweight by using native webviews and Rust instead of bundling Chromium.

Noverterm vs Termius

Noverterm focuses on a native desktop workflow with source-available code and local-first SSH tooling.

Noverterm vs Tabby

Noverterm targets SSH, SFTP, tunnels, keys, and snippets without the weight of an Electron runtime.

FAQ

Questions developers search for.

Is Noverterm free?

Yes for personal and internal company use under the Business Source License 1.1.

Does Noverterm support Windows?

Yes. Noverterm targets macOS and Windows, with Linux support planned.

Does Noverterm support SFTP?

Yes. It includes SFTP browsing, transfers, conflict handling, and file operations.
